### Website Features
- Responsive design suitable for all devices
- Bilingual content (Hindi and English)
- School information and facilities
- Class-wise details from Nursery to 8th
- Contact form for inquiries
- Modern and professional layout

### Sections Included
1. **Home** - Welcome banner with school introduction
2. **About Us** - School features and class information
3. **Classes** - Detailed information about all classes (Nursery to 8th)
4. **Facilities** - School facilities like library, computer lab, playground
5. **Admission** - Admission information and requirements
6. **Contact** - Contact form and school details

### Technical Details
- Built with HTML5, CSS3, Bootstrap
- Responsive design
- Font Awesome icons
- Owl Carousel for image sliders
- Modern CSS animations and effects

### How to Use
1. Open `index.html` in a web browser
2. Navigate through different sections using the menu
3. Contact form can be integrated with backend for functionality
4. Images can be replaced with actual school photos

### Customization
- Replace placeholder images in `assets/images/` with actual school photos
- Update contact information as needed
- Modify content in Hindi/English as required
- Add more sections if needed
